<h1>Description:</h1>
The `gendiff` utility finds differences between files and outputs them in 3 
formats.


## Installation:
```
$ pip install --user git+https://github.com/Gas159/python-project-lvl2.git
```
<hr/>

## Usage:
<p>Running:</p>

```
$ gendiff --format path/to/file1 path/to/file2
```
